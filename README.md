# 糖日記

## 動機
近年來，糖尿病趨向年輕化，患病人數持續增加。糖尿病患血糖如果沒控制好會容易引發很多併發症，所以需要時常注意自己的血糖。一般病患以侵入式採血方式偵測，需要時常更換試紙及採血針，造成醫療耗材的大量使用及成本增加。因此我們想做一個結合非侵入式儀器的app，幫助患者監控血糖，注意自己的健康裝況。
